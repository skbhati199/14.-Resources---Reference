Run Prerequisite Applications With Docker

  1. Install Docker and Docker Compose

  2. Download file Microservice-Pattern-Docker-Scripts.zip provided on this lecture resource

  3. Extract the zip file. Inside of it, you will find folder which has :

    1. file docker-compose.yml, docker-compose-tracking.yml

    2. folder elk/logstash/pipeline, contains file *****.conf

    3. folder mysql/config, contains file *****.cnf

    4. folder mysql/init, contains file *****.sql

  4. From terminal / console, go to directory where zip file extracted (folder Microservice-Pattern-Docker-Scripts)

  5. From terminal / console, run the docker compose script by typing docker-compose up -d

  6. Wait few moments until all containers started

  7. Check the containers, by running docker-compose ps -a from terminal / console. Make sure all containers are running


Java Source Code

  1. Source codes are written using Gradle and Spring Boot.

  2. Download the source code Microservice-Pattern-Docker-Scripts.zip

  3. Extract the zip file. You will found several folders extracted (e.g. employee-movement, industrial-relation, etc). Each folder is a Gradle project.

  4. To run the code, use your favourite Java IDE (e.g. Eclipse, IntelliJ, Netbeans), then import each of gradle project. In this course I use Eclipse.

  5. Then run the gradle project from IDE.


Postman Collection

  1. For the REST API, make sure you have Postman installed

  2. Download file Microservice-Pattern-Postman-Collection.zip

  3. Extract it, you will find file Microservice-Pattern-Postman-Collection

  4. From Postman, click menu File > Import.

  5. Select the extracted postman collection file and import it


Run Hashicorp Vault

  1. Download Microservice-Pattern-Vault.zip from this lecture resource

  2. Extract the zip file into same folder as vault executable (download it from vault official website). Iniside it, you will find file config.hcl

  3. From terminal / console, go to directory where vault executable and config.hcl file located

  4. Use vault server -config=config.hcl to start vault


Run Elastic Stack With Docker

  1. From terminal / console, go to directory where Microservice-Pattern-Docker-Scripts.zip file extracted

  2. From terminal / console, run the docker compose script by typing docker-compose -f docker-compose-tracking.yml up -d

  3. Wait until all containers started

  4. Check the containers, by running docker-compose ps -a from terminal / console. Make sure all containers for elastic stack (elasticsearch, logstash, kibana) are running


[Optional] Cleaning Docker

  1. [Optional] To remove all containers (e.g. when you're done with the course), from terminal/console, go where the docker scripts located and use

    1. docker-compose down or

    2. docker-compose -f docker-compose-tracking.yml down

  2. [Optional] To remove all docker unused images use docker image prune -a
